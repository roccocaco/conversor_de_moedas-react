# Conversor de Moedas (react)

Este é um conversor de moedas simples que permite converter valores entre diferentes moedas em tempo real.

## Funcionalidades

* Conversão entre múltiplas moedas.
* Atualização dinâmica das taxas de câmbio.
* Interface intuitiva e responsiva.

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* API de taxas de câmbio

## Como Usar

1. Selecione a moeda de origem.
2. Selecione a moeda de destino.
3. Insira o valor a ser convertido.
4. Clique no botão de conversão para ver o resultado.
